# 项目简介
## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频、讲解视频、开发工具安装包
- 用户个人信息浏览与操作：个人中心、修改密码、门票订单、门票退订、客房预订、客房退订、车票订单、旅游攻略、我的发布、我的收藏
- 管理员权限操作：用户管理、景点类型管理、景点信息管理、门票订单管理、门票退订管理、室房类型管理、客房信息管理、客房预订管理、客房退订管理、车票类型管理、车票信息管理、车票订单管理、旅游攻略管理、敏感词管理、旅游交流、论坛分类管理、举报记录管理、系统管理、个人中心查看、修改、删除
## 适用场景
- 旅游攻略分享
- 旅游信息查询
- 旅游预订管理
- 旅游交流互动
## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
